# AI Tourism OS — Admin Panel

AI Tourism OS Admin Panel. Internal tool for platform management.
Connects to backend at http://localhost:8000.
Use React Router, TanStack Query, Zustand for auth state.
Only ADMIN and SUPER_ADMIN users can log in.
Simple, functional UI — white background, green (#1A6B4A) primary colour, no frills.
Runs at localhost:3001 (set PORT=3001 in .env).
